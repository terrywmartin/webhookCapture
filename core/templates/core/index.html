Unfortunately, lacking the expertise to build such a tool at the time and with the development team occupied elsewhere, I took matters into my own hands. I crafted a straightforward API endpoint using Node.js, tailored to meet my specific needs.

The primary function of this app is to seamlessly capture webhooks, or any JSON payload for that matter. In my case, the tool proved invaluable for troubleshooting purposes, allowing me to efficiently collect and analyze webhooks for swift issue resolution.

The app itself is very simple.

  1. Create an account.
  2. Create a unique endpoint.
  3. Configure an application to send the webhook to the endpoint.
  4. Watch the requests come in.

Very simple. The payloads are removed when you log out.

Tech Stack:

  • Django
  • Postgresql
  • htmx to give some features a SPA feel
