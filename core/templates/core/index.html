Welcome to a tool I wish I had at my last job. Seriously. This would have been useful. But I didn't have the knowledge to build a tool like this and the dev team was busy doing other things. I ended up building a simple API endpoint in Node.js and it met my needs.

This purpose of this app is to simply capture webhooks, well, any JSON payload really, but my use case was to capture webhooks for troubleshooting purposes.

The app itself is very simple.

  1. Create an account.
  2. Create a unique endpoint.
  3. Configure an application to send the webhook to the endpoint.
  4. Watch the requests come in.

Very simple. The payloads are removed when you log out.

Tech Stack:

  • Django
  • Postgresql
  • htmx to give some features a SPA feel
  • NiceAdmin theme by BootstrapMade
